# Lost And Found System

The **Lost and Found System** is a desktop application that helps students report and find lost items on campus. It allows users to post, search, and manage items in a simple and organized way.

## ⚠️ Important

**Java JDK 23.0.2 is required.** Using a different Java version may cause the project to fail to compile or run.

### Download Java JDK 23.0.2
https://www.oracle.com/java/technologies/javase/jdk23-archive-downloads.html

### Installation Tutorial
https://youtu.be/Eh9-txew5r8

> **Note:** The tutorial installs **Java 25**, but for this project you **must install Java JDK 23.0.2** instead.
